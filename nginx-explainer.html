Nginx sits between the browser and your files — it receives requests and decides what to send back.

📦

Web Server

Serves static files (HTML, CSS, images) directly from disk to the browser.

🔀

Reverse Proxy

In production, forwards requests to your app (Flask, Node) and returns the response.

⚖️

Load Balancer

Splits traffic across multiple app instances so no single one gets overwhelmed.

🔒

SSL Termination

Handles HTTPS encryption so your app code doesn't have to deal with it.

What just happened when you opened localhost

# 1. You typed in browser:
http://localhost

# 2. Browser sent this HTTP request to port 80:
GET / HTTP/1.1
Host: localhost

# 3. Nginx received it, looked at its config:
root /var/www/html ← where to look for files
index index.html ← default file to serve

# 4. Found /var/www/html/index.html → sent it back
200 OK → your dark themed app appeared ✅
